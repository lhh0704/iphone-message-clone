챌린지 중 힘들거나 어려운 부분이 생겼을 때 해결한 방법은?

처음엔 너무 답답하고 힘들었지만 밤을 새가며 공부했고 과제도 만족할 때까지 계속해서 고민하며 작품을 만들었습니다.
그래서 플레이리스트 과제를 만들 때 개인적으로 너무너무 제 작품에 만족해서 하루종일 그것만 감상하기도 했네요.
그리고 저는 복습에 집중했습니다.
진도를 빨리 나가지 못하더라도 배운 내용은 꼭 머릿속에 깊숙이 넣자 라는 마인드로 했던 것 같습니다.

코코아클론을 끝내면서 느낀점은?

HTML, CSS 모두 처음이라 생소하고 어려웠지만 머릿속으로 상상하기만 하던 웹 페이지 장식을 직접 해보니 너무 좋았습니다.
특히 CSS로 하나하나 디자인해가며 웹 페이지를 꾸밀 때의 성취감은 정말 좋은 것 같습니다.
앞으로 자바 스크립트도 잘 배워서 더 똑똑한 웹 페이지를 만들어야겠다고 생각하는 계기가 되었습니다.
